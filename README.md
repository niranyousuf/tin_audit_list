# tin_audit_list

TIN Audit checker web app for NBR AY 2023-24 risk-based audit selection.

## Live Demo

- [https://tin-audit-list.vercel.app/](https://tin-audit-list.vercel.app/)

## Features

- Real-time TIN prefix search suggestions (top 10)
- Exact match audit status check for 12-digit TIN
- Shows `zone`, `circle`, and `submission type`
- Mobile-responsive clean UI
- Structured dataset from official published list

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Plain CSS (no Tailwind)

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
